System Architecture & ML Pipeline

ALGORITHM

Random Forest Classifier

The core engine uses an ensemble learning method. Instead of relying on one "Decision Tree," it creates a forest of multiple trees and takes the majority vote. This prevents "overfitting" and makes our loan predictions highly accurate.

[Image of random forest algorithm diagram]

Feature Engineering

We process 11 features including Applicant Income, Credit History, and Property Area. Categorical data is transformed using Label Encoding to convert text into machine-readable numbers.

SQLite Persistence

The application is full-stack. Every prediction made is captured in a SQL database via SQLAlchemy, allowing for audit trails and historical performance tracking.

Flask Backend

A lightweight Python WSGI server handles the routing, connecting the Scikit-Learn model to the frontend User Interface in real-time.

The Prediction Pipeline

Input Data → Label Encoding → Random Forest Voting → Softmax Probability → UI Visualization
